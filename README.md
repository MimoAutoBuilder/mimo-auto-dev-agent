# 个人项目自动化开发与维护 Agent

基于 Cursor、豆包搭建的多Agent协作开发工具链，后续将全面接入 MiMo 系列模型进行长期迭代与适配。

## 核心功能
- 代码生成Agent：自动生成CRUD业务逻辑、接口定义与单元测试用例
- 重构优化Agent：识别冗余逻辑、优化代码结构，提升可读性与性能
- 文档维护Agent：根据代码变更自动更新项目文档，确保一致性

## 技术栈
- 开发工具：Cursor
- AI模型：GPT / 豆包 / MiMo（规划中）

## 已实现效果
- 已用该工作流完成3个个人项目
- 代码编写效率提升约60%
- 重构与文档维护时间降低近50%

## MiMo 适配规划
1.  适配国内云服务与开源组件，打造更贴合国内开发者的自动化流程
2.  利用MiMo长链推理能力，优化复杂业务场景的跨模块代码生成
3.  新增批量项目管理与CI/CD集成能力，实现项目全生命周期自动化
